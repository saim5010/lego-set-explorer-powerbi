# LEGO Set Explorer – Power BI Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to explore LEGO sets based on themes, categories, price, and piece counts. The dashboard allows users to filter and analyze LEGO sets dynamically while viewing detailed information and images of each set.

## Dashboard Features

* **KPI Cards**

  * Total LEGO Sets
  * Average Pieces per Set
  * Average Price

* **Interactive Filters**

  * Theme Group
  * Theme
  * Age Range
  * Maximum Price

* **LEGO Set Table**
  Displays detailed information including:

  * Set Name
  * Set ID
  * Theme
  * Age Range
  * Number of Pieces
  * Price
  * Price Range

* **Dynamic Set Details Panel**
  When a user selects a LEGO set, the dashboard displays:

  * Set image
  * Set name
  * Price
  * Release year
  * Piece count
  * Recommended age

* **Set Explorer Page**
  Uses a **Decomposition Tree** to analyze:

  * Category → Theme Group → Theme → Set Name

## Tools Used

* Power BI Desktop
* Data Modeling
* DAX Measures
* Interactive Visualizations

## Key Insights

* Licensed themes such as **Star Wars** contain a large number of sets.
* Some LEGO sets exceed **10,000 pieces** and cost more than **$600**.
* Themes and categories significantly influence the number of available sets.
